## Picking the right node

- Need the map: use Hapa Awesome, then Hapa.
- Need canon: use the Worldbuilding Wiki, Lore Node, Second Brain, and Lance.
- Need task state: use Overwatch, Quest Keeper, Overwatch Kanban, or Open Tasks.
- Need runtime status: use Telemetry Node.
- Need primary app behavior: use Hapa Dev Proto.
- Need a 3D/operator scene: use Hapa Space.
- Need model/media generation: use MLX Station, LLaDA, MTPLX, Drama, LiTo, Avatar, Songs, LuminaStem, or Cymatica.
- Need agents: use Hermes and Agent Registry.
- Need trust or secrets: use Keys and Crypto.
- Need cards/protocol artifacts: use Anvil, Consul, Cultivation Suite, and Spec Scaffold.
